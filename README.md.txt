This project analyzes pricing and location trends for Airbnb listings in Mexico City (CDMX) to understand key patterns in pricing, location, reviews, and more.

Objective
Explore and model Airbnb listing data to answer:
What factors most influence nightly prices?

Files
/data/listings.csv: Data downloaded from Inside Airbnb

/notebook/airbnb_cdmx_analysis.ipynb: Complete analysis notebook

Technologies
Python 3

Jupyter Notebook

Pandas, Matplotlib, Seaborn, Scikit-learn

How to Use
Clone this repository

Open the notebook with Jupyter

Ensure required packages are installed (requirements.txt)

Key Findings
The most expensive neighborhoods are concentrated in areas like Condesa and Polanco, while the number of reviews isn't always proportional to price.
